ApodizationFunctions.BartlettApodization — Type
ApodizationFunctions.BartlettApodization(sz; kwargs...)

Create a N-dimensionnal apodization function ApodizationFunctions.BartlettApodization, with size sz defined as a single Int (1-dimensionnal) or a tuple of Int.

For kwargs definitions see bartlett function in package DSP.jl.

Example

julia> apod = ApodizationFunctions.BartlettApodization((4, 4, 4));

julia> A = rand(4, 4, 4);

julia> apod(A)
4×4×4 Array{Float64, 3}:
...
